## Study Rule

- 정해진 분량을 각자 공부 / 정리한다.
    - ppt / notion / blog 등 편한 방법으로 정리
- 매주 목요일에 정리한 내용을 공유(발표)한다.
- 매주 문서 담당자는 공유한 내용을 정리하여 GitHub에 올린다.

## 문서 업로드 규칙

- 개인 공부는 주차별 폴더에 자유롭게 정리해서 공유한다.
- 문서 담당자는 주차별 폴더에 README.md에 정리하여 작성한다.
